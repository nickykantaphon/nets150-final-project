Class Description

ESPNParser - This class takes care of all scraping data from ESPN using JSoup.
Player - The players' information is stored as an object of this class.
DataExtraction - This class extracts data from ESPN by creating an object of ESPNParser class to scrape data from ESPN. After extracting data, this class also constructs a graph in the adjacency matrix form and two associated lists containing name and type of the nodes.
ParserMain - This is the main Class where users can interact with a graph by creating an object of DataExtraction class.
